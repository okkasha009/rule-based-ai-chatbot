# Rule-Based AI Chatbot

## Objective

This chatbot simulates basic human conversation using deterministic rules instead of machine learning.

## Technologies

- Python

- Object-Oriented Programming

- Dictionaries

- Functions

- Loops

- Modular Programming

## Features

- Rule-based responses

- Infinite chat loop

- Exit commands

- Date & Time

- Typing animation

- Colored console

- Random responses
